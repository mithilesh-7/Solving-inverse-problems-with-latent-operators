# Solving-inverse-problems-with-latent-operators project

Designed a SILO+ latent operator with multi-scale convolutions and spatial attention for inpainting and Gaussian deblurring on FFHQ while keeping the Stable Diffusion VAE/UNet frozen.
Applied latent-space measurement consistency, DDPM sampling, AdamW, and EMA weights; achieved lower perceptual error (PSNR, LPIPS) than the SILO baseline across reported tasks.
